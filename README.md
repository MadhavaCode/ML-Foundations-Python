# ML Foundations Python

Core Python data science toolkit including NumPy, Pandas, Matplotlib, Seaborn with worked examples and real-dataset visualizations.

## 📓 Notebooks

| Notebook | Topics Covered |
|---|---|
| `Numpy.ipynb` | Arrays, reshaping, slicing, broadcasting, math ops (sin/cos/log/exp), statistics |
| `Pandas.ipynb` | DataFrames, Series, filtering, groupby, merging, data cleaning |
| `MatPlotLib.ipynb` | Line/scatter/bar charts, subplots, axis labels, legends |
| `MatPlotLib-HW.ipynb` | Applied exercises on real advertising and sales datasets |
| `Seaborn.ipynb` | countplot, boxplot, scatterplot, pairplot, heatmap on real data |

## 🛠 Tech Stack
`Python` · `NumPy` · `Pandas` · `Matplotlib` · `Seaborn`
